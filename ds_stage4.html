Step 4: Visualize Data with Matplotlib and Seaborn

In this stage you learn how to turn numbers into simple, clear charts using Matplotlib and Seaborn in Python.

Good visualizations help you quickly see trends, compare groups, and explain results to others, which is a core part of any data science job.


1Ô∏è‚É£ Setup ‚Äì import libraries and simple data

First, make sure you have Matplotlib and Seaborn installed, then create a small dataset (for example student marks) directly in code. 

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# optional: nicer default style
sns.set_style("whitegrid")

data = {
    "name": ["Asha", "Ravi", "Meena", "Sita", "Kiran"],
    "maths": [60, 75, 80, 90, 70],
    "science": [65, 70, 85, 88, 72]
}
df = pd.DataFrame(data)
print(df.head())

Using a small DataFrame keeps things simple for beginners while still looking like real data from a class or college.

2Ô∏è‚É£ Line plot and bar chart with Matplotlib

Matplotlib gives low‚Äëlevel control to build basic charts like line plots and bar charts. 

# Line plot of maths marks
plt.figure(figsize=(6, 4))
plt.plot(df["name"], df["maths"], marker="o", color="blue")
plt.xlabel("Student")
plt.ylabel("Maths Marks")
plt.title("Maths Marks of Students")
plt.tight_layout()
plt.show()

# Bar chart of science marks
plt.figure(figsize=(6, 4))
plt.bar(df["name"], df["science"], color="green")
plt.xlabel("Student")
plt.ylabel("Science Marks")
plt.title("Science Marks of Students")
plt.tight_layout()
plt.show()

Labels, titles, and tight_layout() make charts easier to read and ready to show to teachers, friends, or in mini‚Äëprojects.

3Ô∏è‚É£ Scatter plot and distribution plot with Seaborn

Seaborn sits on top of Matplotlib and makes it easier to create attractive statistical plots with less code. 

# Scatter plot: maths vs science
plt.figure(figsize=(6, 4))
sns.scatterplot(data=df, x="maths", y="science")
plt.xlabel("Maths Marks")
plt.ylabel("Science Marks")
plt.title("Maths vs Science Marks")
plt.tight_layout()
plt.show()

# Distribution of maths marks
plt.figure(figsize=(6, 4))
sns.histplot(data=df, x="maths", kde=True)
plt.xlabel("Maths Marks")
plt.title("Distribution of Maths Marks")
plt.tight_layout()
plt.show()

Scatter plots show relationships between two numeric variables, while histogram or distribution plots show how values are spread out.

4Ô∏è‚É£ Simple grouped bar chart with Seaborn

You can also compare categories using bar plots, for example average marks by subject or branch. 

# Convert to long format for easier plotting
df_long = df.melt(id_vars="name",
                  value_vars=["maths", "science"],
                  var_name="subject",
                  value_name="marks")

plt.figure(figsize=(7, 4))
sns.barplot(data=df_long, x="name", y="marks", hue="subject")
plt.xlabel("Student")
plt.ylabel("Marks")
plt.title("Maths vs Science Marks by Student")
plt.tight_layout()
plt.show()

Many tutorials use this ‚Äúlong‚Äù format plus hue to compare multiple groups in one clean chart.
