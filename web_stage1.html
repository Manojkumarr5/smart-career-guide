Step 1: Understand HTML Structure and Semantics

This step starts from absolute zero. Imagine you have never seen code before. You will learn what an HTML file is, what each line means, and how the browser reads it.


1Ô∏è‚É£ What is a website and a web page?

A website is a collection of web pages. A web page is what you see in your browser when you open a link like https://example.com.

Each web page is usually built using three main technologies: HTML (structure), CSS (design), and JavaScript (behavior). In this step, you focus only on HTML.

2Ô∏è‚É£ What is HTML?

HTML means HyperText Markup Language. It is not a programming language. It is a way to tell the browser ‚Äúthis is a heading, this is a paragraph, this is a link, this is an image‚Äù.

HTML uses tags. A tag looks like this: <p> ... </p>. The opening tag starts an element, the closing tag ends it, and inside you write your content.

4Ô∏è‚É£ Explanation of each line

Now see what each line in the above example means, in simple words.

  • <!DOCTYPE html> ‚Üí This tells the browser ‚Äúthis document is written in HTML5‚Äù (modern HTML). It should always be the first line.
  • <html lang="en"> ‚Üí This is the root of the page. Everything else goes inside <html> and </html>. The attribute lang="en" says the language of the page is English.
  • <head> ... </head> ‚Üí The head section holds information about the page which is not shown directly, like the title, character settings, and links to CSS.
  • <meta charset="UTF-8"> ‚Üí This line tells the browser which character set to use. UTF-8 allows almost all languages and symbols to display correctly.
  • <title>My First Web Page</title> ‚Üí This sets the title that appears on the browser tab and in bookmarks. It does not appear inside the main page area.
  • <body> ... </body> ‚Üí The body contains everything that the user can actually see: headings, text, images, buttons, etc.
  • <h1>Hello, World!</h1> ‚Üí This is the main big heading of the page. Usually used for the most important title.
  • <p>This is my first HTML page.</p> ‚Üí This is a paragraph of normal text.
  • </html> ‚Üí This closes the html tag and ends the document.

5Ô∏è‚É£ More headings and paragraphs

You can use more headings to create sections. h1 is biggest, h6 is smallest. Paragraphs are for regular text.

7Ô∏è‚É£ Semantic layout: header, nav, main, section, footer

Semantic elements give meaning to parts of your page. They help search engines and screen readers understand the structure.

  • <header> ‚Üí Top part of the page (logo, title, navigation).
  • <nav> ‚Üí Navigation menu with links.
  • <main> ‚Üí Main content for this page.
  • <section> ‚Üí A logical section of content (like ‚ÄúCourses‚Äù or ‚ÄúContact‚Äù).
  • <footer> ‚Üí Bottom part of the page (copyright, extra links).
