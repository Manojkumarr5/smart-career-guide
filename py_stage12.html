2Ô∏è‚É£ File and folder automation basics

You already know os from earlier stages; now you use it to build real scripts that operate on many files at once. 

import os

folder = "downloads_sample"   # change to your folder name

for name in os.listdir(folder):
    path = os.path.join(folder, name)
    if os.path.isfile(path):
        print("File:", path)
    else:
        print("Folder:", path)

Listing files is often the first step before renaming, moving, or analyzing them.

3Ô∏è‚É£ Script: simple file organizer

This script organizes files into subfolders by extension (for example, .pdf, .jpg, .py). 

import os
import shutil

folder = "downloads_sample"   # your folder
folder = os.path.abspath(folder)

# mapping extension to folder name
mapping = {
    ".pdf": "PDFs",
    ".jpg": "Images",
    ".png": "Images",
    ".jpeg": "Images",
    ".py": "Python",
    ".txt": "Text",
}

for name in os.listdir(folder):
    path = os.path.join(folder, name)

    if not os.path.isfile(path):
        continue

    _, ext = os.path.splitext(name)
    ext = ext.lower()

    target_dir = mapping.get(ext, "Others")
    target_path = os.path.join(folder, target_dir)

    os.makedirs(target_path, exist_ok=True)

    new_location = os.path.join(target_path, name)
    print("Moving:", path, "->", new_location)
    shutil.move(path, new_location)

This is a classic real‚Äëworld automation: tidying up a messy downloads folder with one script instead of manual drag‚Äëand‚Äëdrop.

4Ô∏è‚É£ Script: batch rename files

Another common automation is renaming many files in a pattern, for example, project_1.jpg, project_2.jpg, and so on. 

import os

folder = "images_sample"
folder = os.path.abspath(folder)

base_name = "project"
counter = 1

for name in os.listdir(folder):
    path = os.path.join(folder, name)
    if not os.path.isfile(path):
        continue

    root, ext = os.path.splitext(name)
    new_name = f"{base_name}_{counter}{ext}"
    new_path = os.path.join(folder, new_name)

    print("Renaming:", path, "->", new_path)
    os.rename(path, new_path)
    counter += 1

Always test batch scripts in a small test folder first, not on important files.

5Ô∏è‚É£ Script: quick report generator

You can combine file handling and basic calculations to create small local reports, such as counting how many files of each type you have. 

import os
from collections import Counter

folder = "downloads_sample"
folder = os.path.abspath(folder)

counter = Counter()

for name in os.listdir(folder):
    path = os.path.join(folder, name)
    if os.path.isfile(path):
        _, ext = os.path.splitext(name)
        counter[ext.lower()] += 1

print("File counts by extension:")
for ext, count in counter.items():
    print(ext or "(no extension)", ":", count)

This style (scan folder ‚Üí count or aggregate ‚Üí print summary) appears in many small company and student scripts.
