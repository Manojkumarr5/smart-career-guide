Step 5: Perform Exploratory Data Analysis (EDA)

In this stage you learn how to ‚Äúexplore‚Äù a dataset: understand its shape, distributions, relationships, and unusual values before building any model.

EDA is like doing a health check of your data using summaries, plots, and good questions, so you can later choose better features and models.


1Ô∏è‚É£ Start with questions and load the data

Good EDA begins by asking simple questions such as ‚ÄúWhat does each column mean?‚Äù and ‚ÄúWhat do we want to find out from this dataset?‚Äù.

  • Example goal: ‚ÄúUnderstand how student branch and study hours relate to marks.‚Äù
  • Example questions: ‚ÄúWhat is the average mark?‚Äù, ‚ÄúDo some branches score higher?‚Äù, ‚ÄúAre there any extreme values?‚Äù.
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

sns.set_style("whitegrid")

df = pd.read_csv("students_clean.csv")
print(df.head())
print(df.info())
print(df.describe())

info() shows data types and missing values, while describe() gives basic descriptive statistics for numeric columns.

2Ô∏è‚É£ Understand distributions (single variable)

First look at each important numeric column alone to see its range, center, and spread using histograms or box plots. 

# Histogram of marks
plt.figure(figsize=(6, 4))
sns.histplot(data=df, x="marks", kde=True)
plt.xlabel("Marks")
plt.title("Distribution of Marks")
plt.tight_layout()
plt.show()

# Box plot of marks
plt.figure(figsize=(4, 5))
sns.boxplot(y="marks", data=df)
plt.title("Marks ‚Äì Box Plot")
plt.tight_layout()
plt.show()

Histograms and box plots help you spot skewed data and outliers (very high or very low points compared to most of the class).

3Ô∏è‚É£ Explore relationships (two variables)

Next, check how two variables move together: for example, study hours vs marks, or attendance vs marks. 

# Scatter plot: study_hours vs marks
plt.figure(figsize=(6, 4))
sns.scatterplot(data=df, x="study_hours", y="marks")
plt.xlabel("Study Hours")
plt.ylabel("Marks")
plt.title("Study Hours vs Marks")
plt.tight_layout()
plt.show()

# Average marks by branch (categorical vs numeric)
plt.figure(figsize=(6, 4))
sns.barplot(data=df, x="branch", y="marks", estimator="mean")
plt.xlabel("Branch")
plt.ylabel("Average Marks")
plt.title("Average Marks by Branch")
plt.tight_layout()
plt.show()

Looking at relationships highlights patterns like ‚Äúmore hours ‚Üí higher marks‚Äù or differences between groups such as branches or sections.

4Ô∏è‚É£ Spot outliers and simple data issues

EDA also includes checking for strange values (like marks above 100) or records that do not make sense, which may indicate data quality problems. 

# Quick rule-based outlier check
outliers = df[df["marks"] > 100]
print("Rows with marks > 100:")
print(outliers)

# You can also look at the top/bottom values
print(df["marks"].sort_values().head())
print(df["marks"].sort_values(ascending=False).head())

When you find outliers, decide whether they are data entry mistakes (to fix or remove) or real extreme cases that must be kept and explained.
