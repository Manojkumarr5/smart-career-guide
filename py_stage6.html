Step 6: Explore Useful Libraries ‚Äì os, math, and random

In this stage you will meet three very useful standard libraries: os for talking to your operating system (files, folders), math for extra mathematical functions, and random for random numbers and choices.

These come built‚Äëin with Python, so you can use them without installing anything, and they will appear again in many projects (automation, games, simulations, data work).


1Ô∏è‚É£ Using modules with import

A module is just a Python file with extra functions and tools. To use it, you need to import it.

Basic patterns:

import math           # import whole module
import os             # import whole module
import random         # import whole module

# or
from math import sqrt  # import only sqrt function

At this stage, it is fine to use simple import math, import os, import random and then call functions as math.sqrt(), os.getcwd(), random.randint().

2Ô∏è‚É£ math module ‚Äì extra maths power

The math module gives you many functions for more serious math: square root, powers, trigonometry, rounding, and more. 

import math

x = 25
print("Square root of", x, "is", math.sqrt(x))

angle_deg = 30
angle_rad = math.radians(angle_deg)  # convert degrees to radians
print("sin(30¬∞) =", math.sin(angle_rad))

print("Floor of 3.7 =", math.floor(3.7))
print("Ceil of 3.1 =", math.ceil(3.1))

These are helpful for physics formulas, engineering calculations, and data analysis where you need more than just +, -, *, /.

3Ô∏è‚É£ random module ‚Äì random numbers and choices

The random module generates pseudo‚Äërandom numbers that are good enough for games, simulations, practice questions, and more. 

import random

# random integer between 1 and 6 (like a dice)
dice = random.randint(1, 6)
print("Dice rolled:", dice)

# random float between 0 and 1
r = random.random()
print("Random float between 0 and 1:", r)

# choose a random item from a list
students = ["Asha", "Ravi", "Kiran", "Meera"]
winner = random.choice(students)
print("Randomly selected student:", winner)

There are more functions like random.shuffle() to shuffle a list and random.sample() to pick several unique items.

4Ô∏è‚É£ Small game ‚Äì number guessing

Here is a simple guessing game using random.randint(). 

import random

secret = random.randint(1, 20)
print("I am thinking of a number between 1 and 20.")

for attempt in range(1, 6):
    guess = int(input("Attempt " + str(attempt) + ": Enter your guess: "))

    if guess < secret:
        print("Too low!")
    elif guess > secret:
        print("Too high!")
    else:
        print("Correct! You guessed it.")
        break

if guess != secret:
    print("Sorry, the number was:", secret)

This pattern (random + loop + if/else) is used in many beginner games and practice tools.

5Ô∏è‚É£ os module ‚Äì working with folders and files

The os module lets Python talk to your operating system. You can check which folder you are in, list files, create folders, and more. 

import os

# current working directory
print("Current folder:", os.getcwd())

# list files and folders in current directory
items = os.listdir()
print("Items here:", items)

# make a new folder (if not exists)
if "my_notes" not in items:
    os.mkdir("my_notes")
    print("Created folder my_notes")

Be careful with functions like os.remove() and os.rmdir() because they delete files or folders. Use them only when you are sure.

6Ô∏è‚É£ Combining os with file handling

You can combine os with the file handling you learned in Stage 4 to make small automation scripts. 

import os

folder = "logs"

# create folder if needed
if not os.path.exists(folder):
    os.mkdir(folder)

filepath = os.path.join(folder, "today_log.txt")

with open(filepath, "a") as f:
    f.write("Student opened the app.\n")

print("Log updated at:", filepath)

os.path.join() helps build correct paths in a cross‚Äëplatform way (works on Windows, macOS, Linux).
