Step 9: Visualize Data with Matplotlib

In this stage you learn how to create basic graphs using Matplotlib, the most common plotting library in Python. You will make simple line, bar, and scatter plots to visualize trends and comparisons.

These skills help you turn raw numbers into pictures so that students, teachers, or managers can understand the data at a glance.


1Ô∏è‚É£ Installing and importing Matplotlib

If Matplotlib is not installed yet, you can install it with pip.

# install once
pip install matplotlib

Then at the top of your Python file:

import matplotlib.pyplot as plt

The pyplot module (imported as plt) is the part of Matplotlib used for most basic plots.

2Ô∏è‚É£ Basic line plot

A line plot is useful to show how a value changes over time or steps. 

import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]          # x-axis values (for example, weeks)
y = [10, 14, 12, 18, 20]     # y-axis values (for example, hours studied)

plt.plot(x, y, marker="o")   # line with circle markers

plt.title("Study Hours Over 5 Days")
plt.xlabel("Day")
plt.ylabel("Hours studied")

plt.grid(True)
plt.show()

plt.show() displays the graph, and plt.grid(True) adds a helpful background grid.

3Ô∏è‚É£ Bar chart ‚Äì comparing categories

Bar charts are great for comparing values between different groups, such as marks of subjects or number of students in branches. 

import matplotlib.pyplot as plt

subjects = ["Maths", "Python", "DBMS", "ML"]
marks = [78, 88, 75, 92]

plt.bar(subjects, marks, color=["#4caf50", "#2196f3", "#ff9800", "#9c27b0"])

plt.title("Marks by Subject")
plt.xlabel("Subject")
plt.ylabel("Marks")

plt.ylim(0, 100)  # y-axis from 0 to 100
plt.show()

You can change colors to make charts more readable, especially when presenting to others.

4Ô∏è‚É£ Scatter plot ‚Äì relationship between two variables

Scatter plots help you see the relationship between two variables, for example, hours studied vs marks scored. 

import matplotlib.pyplot as plt

hours = [1, 2, 3, 4, 5, 6]
marks = [40, 50, 60, 70, 80, 90]

plt.scatter(hours, marks, color="#2196f3")

plt.title("Study Hours vs Marks")
plt.xlabel("Hours studied")
plt.ylabel("Marks")

plt.grid(True)
plt.show()

In this example, the points roughly form an increasing line, suggesting that more hours studied usually leads to higher marks.

5Ô∏è‚É£ Adding multiple lines in one graph

You can show multiple series in one plot using multiple plt.plot() calls along with a legend. 

import matplotlib.pyplot as plt

days = [1, 2, 3, 4, 5]

student_a = [2, 3, 4, 3, 5]
student_b = [1, 2, 2, 3, 4]

plt.plot(days, student_a, marker="o", label="Student A")
plt.plot(days, student_b, marker="s", label="Student B")

plt.title("Daily Study Hours ‚Äì Two Students")
plt.xlabel("Day")
plt.ylabel("Hours studied")

plt.legend()      # show labels
plt.grid(True)
plt.show()

Legends are important when you have more than one line so the viewer knows which line represents which series.

6Ô∏è‚É£ Saving plots to image files

Instead of only showing the chart, you can also save it as a PNG image and then use it in slides or reports. 

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 15, 30]

plt.plot(x, y, marker="o")
plt.title("Sample Plot")

plt.savefig("sample_plot.png")   # save to file
plt.show()

The file is saved in the current working directory; you can pass a full path if needed.

7Ô∏è‚É£ Simple workflow with pandas + Matplotlib

In many real projects, you will first load data with pandas and then visualize it with Matplotlib. 

import pandas as pd
import matplotlib.pyplot as plt

# Suppose marks.csv has columns: name, branch, marks
df = pd.read_csv("marks.csv")

# bar chart of marks
plt.bar(df["name"], df["marks"])
plt.title("Student Marks")
plt.xlabel("Student")
plt.ylabel("Marks")
plt.xticks(rotation=45)    # rotate labels for readability
plt.tight_layout()         # adjust layout
plt.show()

Combining pandas and Matplotlib makes it easy to go from raw CSV to charts in just a few lines.
