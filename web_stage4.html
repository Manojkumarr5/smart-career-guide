Step 4: JavaScript Basics (Variables, Loops, Functions)

In Stage 1 and 2 you created structured and styled pages. In Stage 3 you built responsive layouts. Now, in Stage 4, you start adding ‚Äúbrain‚Äù to your website using JavaScript.

JavaScript lets your page react to user actions: click a button, show a message, change text, calculate results, and much more.


1Ô∏è‚É£ What is JavaScript?

JavaScript is a programming language that runs inside the browser. It can read and change HTML elements, handle events like clicks, and perform calculations.

With JavaScript you can build things like calculators, todo lists, quizzes, animations, and full web apps.

2Ô∏è‚É£ How to add JavaScript to a page

There are two main ways to add JavaScript in your project:

  • Inline / internal: <script> tag directly inside your HTML file.
  • External file: a separate .js file linked with <script src="app.js"></script>.

3Ô∏è‚É£ Variables: storing values

Variables are like labeled boxes where you can store values such as numbers, text, or true/false values. In modern JavaScript, you usually use let and const to create variables.

Common data types: numbers (10, 3.14), strings ("Hello"), booleans (true/false).

// Using let (value can change)
let age = 20;
age = 21;  // allowed


// Using const (value cannot change)
const appName = "Smart Career Guide";


// Strings and numbers
let studentName = "Manoj";
let score = 95;


// Boolean
let isLoggedIn = true;

4Ô∏è‚É£ Basic operators and expressions

Operators let you do calculations or comparisons with variables.

  • Arithmetic: + (add), - (subtract), * (multiply), / (divide).
  • Comparison: ==, ===, !=, >, <, etc.
  • Logical: && (and), || (or), ! (not).
let a = 10;
let b = 3;


let sum = a + b;      // 13
let multiply = a * b; // 30


let isGreater = a > b;      // true
let bothTrue = (a > 5) && (b > 5); // false

5Ô∏è‚É£ Functions: reusable blocks of code

A function is a block of code that does a specific task. You define it once and can call it many times. Functions can take input values (parameters) and can give back an output (return value).

Think of a function like a small machine: you give inputs, it processes them, and returns a result.

// Function without parameters
function sayHello() {
  console.log("Hello, welcome to Smart Career Guide!");
}


// Function with parameters
function addNumbers(x, y) {
  const result = x + y;
  return result;
}


// Calling (using) functions
sayHello();
let total = addNumbers(5, 7); // total will be 12

6Ô∏è‚É£ Loops: repeat something many times

Loops let you run the same block of code multiple times. The most common beginner loops are for and while.

Example: count from 1 to 5 and build a message.

// For loop example
let message = "";
for (let i = 1; i <= 5; i++) {
  message = message + "Number: " + i + "\n";
}


// While loop example
let count = 1;
let msg2 = "";
while (count <= 3) {
  msg2 = msg2 + "Count is " + count + "\n";
  count++;
}

8Ô∏è‚É£ Putting it together: small ‚Äúscore calculator‚Äù

This example uses variables, a function, and a loop to calculate the total of an array of marks, and then shows the average.

// Array of marks
const marks = [80, 90, 75, 88];


// Function to calculate sum
function calculateSum(arr) {
  let total = 0;
  for (let i = 0; i < arr.length; i++) {
    total = total + arr[i];
  }
  return total;
}


// Use the function
const totalMarks = calculateSum(marks);
const avgMarks = totalMarks / marks.length;


console.log("Total:", totalMarks);
console.log("Average:", avgMarks);
