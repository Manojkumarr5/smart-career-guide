Step 8: Learn Pandas for Data Analysis

In this stage you start using pandas, a powerful library for working with tabular data (rows and columns) similar to Excel or SQL tables. Pandas makes it easy to load, clean, filter, and analyze data.

Even with basic commands, you can already do useful data work for marks, attendance, simple reports, and small projects.


1Ô∏è‚É£ Installing and importing pandas

Pandas is not in the core Python installation, so install it once using pip.

# install once
pip install pandas

Then in your Python file or notebook:

import pandas as pd

The alias pd is a common short name used in almost all tutorials and real projects.

2Ô∏è‚É£ Key ideas: Series and DataFrame

Pandas has two main building blocks: Series and DataFrame.

  • Series ‚Äì like one column (1‚ÄëD).
  • DataFrame ‚Äì table with rows and columns (2‚ÄëD), made of many Series.
import pandas as pd

# Series example
s = pd.Series([10, 20, 30, 40])
print("Series:")
print(s)

# DataFrame example from a dict
data = {
    "name": ["Asha", "Ravi", "Kiran"],
    "marks": [85, 72, 91],
    "branch": ["CSE", "ECE", "ME"]
}
df = pd.DataFrame(data)

print("\nDataFrame:")
print(df)

In the DataFrame, each key in the dictionary becomes a column.

3Ô∏è‚É£ Loading data from CSV

In real use, you will often read data from a CSV file (comma‚Äëseparated values), and pandas makes this a one‚Äëline operation. 

import pandas as pd

# marks.csv should be in the same folder
df = pd.read_csv("marks.csv")

print("First 5 rows:")
print(df.head())

head() shows the first few rows; use df.tail() for the last rows.

4Ô∏è‚É£ Exploring your DataFrame

Pandas provides quick methods to understand the data shape and basic statistics. 

print("Shape (rows, columns):", df.shape)
print("\nColumn names:", df.columns)

print("\nSummary statistics:")
print(df.describe())    # for numeric columns

print("\nInfo:")
print(df.info())

These commands help you quickly check if the dataset looks correct before analysis.

5Ô∏è‚É£ Selecting columns and rows

You can select specific columns like this: 

# one column (Series)
names = df["name"]

# multiple columns (DataFrame)
marks_and_branch = df[["name", "marks", "branch"]]

Filtering rows based on conditions:

# students with marks >= 80
top_students = df[df["marks"] >= 80]

# CSE students with marks >= 75
cse_good = df[(df["branch"] == "CSE") & (df["marks"] >= 75)]

This pattern df[condition] is extremely powerful for real data cleaning and filtering.

6Ô∏è‚É£ Adding new columns

You can create new columns based on calculations or conditions. 

# suppose marks are out of 100
df["result"] = df["marks"].apply(lambda m: "Pass" if m >= 35 else "Fail")

# add grade
def grade(m):
    if m >= 80:
        return "A"
    elif m >= 60:
        return "B"
    elif m >= 35:
        return "C"
    else:
        return "F"

df["grade"] = df["marks"].apply(grade)

print(df.head())

Functions with apply() let you convert business rules (like grading) into code.

7Ô∏è‚É£ Grouping and summarizing data

Grouping allows you to calculate statistics for each category, for example branch‚Äëwise average marks. 

# average marks by branch
branch_avg = df.groupby("branch")["marks"].mean()
print(branch_avg)

# count of pass/fail
result_counts = df["result"].value_counts()
print(result_counts)

groupby() is a key feature for reports and dashboards.

8Ô∏è‚É£ Saving your cleaned data

After cleaning or adding new columns, you can save the updated DataFrame back to a CSV. 

df.to_csv("marks_cleaned.csv", index=False)

Setting index=False avoids saving the DataFrame index as an extra column.
