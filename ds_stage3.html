Step 3: Learn Data Cleaning Using Pandas

In this stage you learn how to clean messy data using Pandas in Python: loading a CSV, checking for missing values, fixing them, and removing obvious mistakes or duplicates.

Most real‚Äëlife datasets have missing entries, extra spaces, wrong types, or repeated rows, so data cleaning is one of the most important daily skills in data science.

2Ô∏è‚É£ Load data in Pandas and inspect

Now use Pandas to load this CSV and quickly check its structure. 

import pandas as pd

df = pd.read_csv("students_raw.csv")

print(df.head())        # first few rows
print(df.info())        # column types and non-null counts
print(df.isna().sum())  # how many missing values in each column

Functions like head(), info(), and isna().sum() help you see problems such as missing values or wrong data types in seconds.

3Ô∏è‚É£ Handle missing values

There are many ways to handle missing values: drop rows, fill with a fixed value, or fill using statistics like mean or median. 

# Option 1: drop rows where cgpa is missing
df_drop = df.dropna(subset=["cgpa"])

# Option 2: fill missing cgpa with average cgpa
avg_cgpa = df["cgpa"].mean()
df_fill = df.copy()
df_fill["cgpa"] = df_fill["cgpa"].fillna(avg_cgpa)

For beginner projects, clearly explain in comments why you choose to drop or fill missing values so that students understand the trade‚Äëoff.

4Ô∏è‚É£ Fix spaces, text, and duplicates

Raw data often has extra spaces and repeated rows; Pandas gives simple functions to standardize and remove duplicates. 

clean = df.copy()

# Remove extra spaces from text columns
clean["name"] = clean["name"].str.strip()
clean["branch"] = clean["branch"].str.strip()

# Drop exact duplicate rows
clean = clean.drop_duplicates()

Cleaning text (using str.strip(), str.lower(), etc.) and removing duplicates with drop_duplicates() are standard data‚Äëwrangling steps in many projects.

5Ô∏è‚É£ Convert data types if needed

Sometimes numeric columns are accidentally read as text because of symbols or missing values; converting dtypes makes calculations possible. 

# Ensure cgpa is numeric, convert errors to NaN
clean["cgpa"] = pd.to_numeric(clean["cgpa"], errors="coerce")

print(clean.dtypes)

After converting, you can again use isna().sum() to see if any new missing values appeared and decide how to handle them.
