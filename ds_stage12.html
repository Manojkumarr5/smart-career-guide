Step 12: Understand Time Series Forecasting

In this stage you learn the basics of time series and forecasting: data ordered by time (day, month, year) and simple ways to predict future values using past observations.

You will see key ideas like trend and seasonality, then build a tiny baseline forecast in Python so students can practice with data like sales or rainfall.


1Ô∏è‚É£ What is a time series?

A time series is a sequence of values recorded over time, for example daily temperature, monthly sales, or yearly rainfall of a district.

  • Trend: long‚Äëterm upward or downward movement (for example sales growing each year).[web:434][web:440]
  • Seasonality: regular repeating pattern (for example higher sales every festival month, or rainfall every monsoon).[web:433][web:437]

Forecasting guides explain that understanding trend and seasonality helps you choose better methods and avoid treating time‚Äëordered data like ordinary rows.

2Ô∏è‚É£ Prepare a simple time series in Pandas

Time series are usually stored with a date column converted to a datetime type and set as the index so that plotting and resampling become easy. 

import pandas as pd

# Example: monthly sales (toy data)
data = {
    "month": [
        "2024-01-01", "2024-02-01", "2024-03-01", "2024-04-01",
        "2024-05-01", "2024-06-01", "2024-07-01", "2024-08-01",
        "2024-09-01", "2024-10-01", "2024-11-01", "2024-12-01"
    ],
    "sales": [120, 130, 125, 140, 150, 160, 170, 180, 175, 190, 210, 230]
}
df = pd.DataFrame(data)
df["month"] = pd.to_datetime(df["month"])
df = df.set_index("month")

print(df.head())

Many tutorials treat the datetime index as the base structure for plotting, splitting, and forecasting.

3Ô∏è‚É£ Visualize and make a simple baseline forecast

Before using advanced models, start with simple ‚Äúbaseline‚Äù forecasts such as repeating the last value, which is called a naive forecast. 

import matplotlib.pyplot as plt

# Plot the historical series
df["sales"].plot(figsize=(7, 4), marker="o", title="Monthly Sales")
plt.ylabel("Sales")
plt.tight_layout()
plt.show()

# Naive forecast: next month's sales = last observed value
last_value = df["sales"].iloc[-1]
print("Naive forecast for next month:", last_value)

Baseline models help you check whether more complex models actually improve the forecast compared with simply using the last observation.

4Ô∏è‚É£ Train/test split for time series

Unlike random train‚Äëtest splits, time series splits must respect time order: train on earlier data and test on later data. 

# Simple time-based split: last 3 months as test
train = df.iloc[:-3]
test  = df.iloc[-3:]

print("Train period:", train.index.min(), "to", train.index.max())
print("Test period:", test.index.min(), "to", test.index.max())

# Naive forecast on test: always use last train value
naive_forecast = [train["sales"].iloc[-1]] * len(test)
print("Test actual:\\n", test["sales"].values)
print("Naive forecast:\\n", naive_forecast)

Time‚Äëbased splits like this are a first step before more advanced methods such as rolling windows or dedicated time series cross‚Äëvalidation.
