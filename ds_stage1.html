Step 1: Learn Python and Math Fundamentals

In this stage you set up Python on your computer, write your first small programs, and revise the simple math used again and again in data science such as percentages, averages, and basic equations.

Later stages will cover statistics, machine learning, and big data, but here the aim is to feel comfortable with basic coding and school‚Äëlevel math so formulas do not feel intimidating.

3Ô∏è‚É£ Learn basic Python building blocks

Before libraries such as NumPy or Pandas, focus on Python fundamentals: variables, input, arithmetic, and printing results.

  • Create a new file math_demo.py in the same folder.
  • Add a program that asks for two numbers and prints their sum.

b = int(input("Enter second number: "))

print("Total is:", total)

This connects Python with calculations, similar to what you will later do with real data like marks, prices, or sensor readings.

4Ô∏è‚É£ Revise basic math used in data science

Data science relies on everyday math such as averages, percentages, and simple algebra, plus more statistics later.

  • Revise natural numbers, integers, fractions, ratios, and percentages.
  • Practice computing the average (mean) of 3‚Äì5 numbers, like marks of 5 students.
  • Practice computing percentage, for example ‚Äúmarks scored / total marks √ó 100‚Äù.

Now combine this math with Python in a new file marks_calculator.py:

m2 = float(input("Marks in Subject 2: "))
m3 = float(input("Marks in Subject 3: "))

total = m1 + m2 + m3

Change 300 to your actual total marks (for example 500 or 600); this pattern appears again in analytics and reporting.
