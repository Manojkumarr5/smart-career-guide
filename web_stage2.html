Step 2: Master CSS Fundamentals and Colors

In Stage 1, you learned how to build the structure of a page with HTML. Now, in Stage 2, you will learn how to make that page look beautiful using CSS (Cascading Style Sheets).

Think of HTML as the skeleton and CSS as the clothes and style: colors, fonts, spacing, borders, and layout.


1Ô∏è‚É£ What is CSS?

CSS is a language that controls how HTML elements look. It tells the browser things like ‚Äúmake this heading blue‚Äù, ‚Äúadd space here‚Äù, ‚Äúgive this box a border‚Äù, and so on.

CSS works with rules. Each rule has: a selector (what to style) and some properties (how to style it).

/* Example CSS rule */
h1 {
  color: blue;           /* property: value; */
  font-size: 32px;
}

2Ô∏è‚É£ Three ways to add CSS

There are 3 main ways to write CSS for your HTML page:

  • Inline CSS ‚Äì style inside a tag using the style attribute (good for quick tests).
  • Internal CSS ‚Äì CSS inside a <style> tag in the <head> of your HTML.
  • External CSS ‚Äì CSS in a separate .css file, linked from your HTML (best way for real projects).

3Ô∏è‚É£ Basic selectors: element, class, id

Selectors tell CSS which elements to style.

  • Element selector ‚Äì selects all elements with that tag name (for example, all <p> tags).
  • Class selector ‚Äì starts with ., like .card. You can put the same class on many elements.
  • ID selector ‚Äì starts with #, like #mainTitle. IDs should be unique on the page.
/* Element selector: applies to every h1 */
h1 {
  color: #0078d7;
}


/* Class selector: applies to all elements with class="card" */
.card {
  background-color: #ffffff;
  border-radius: 8px;
  padding: 16px;
}


/* ID selector: applies only to the element with id="mainTitle" */
#mainTitle {
  text-align: center;
  font-size: 32px;
}

4Ô∏è‚É£ Common CSS properties

Some basic properties you will use all the time:

  • color ‚Äì text color.
  • background-color ‚Äì background color of an element.
  • font-size ‚Äì size of the text.
  • font-family ‚Äì font type (like Arial, Verdana).
  • margin ‚Äì space outside the element.
  • padding ‚Äì space inside the element, between border and content.
  • border ‚Äì outline around the element.
body {
  font-family: Arial, sans-serif;  /* Font type for all text */
  background-color: #f2f6ff;       /* Page background color */
}

h1 {
  color: #005fa3;                  /* Text color */
  font-size: 32px;                 /* Text size */
}

p {
  color: #333333;
  line-height: 1.6;                /* Space between lines of text */
}

5Ô∏è‚É£ Understanding colors in CSS

There are different ways to write colors in CSS:

  • Color names ‚Äì like red, blue, green.
  • Hex values ‚Äì like #ff0000 (red), #00ff00 (green), #0000ff (blue).
  • RGB ‚Äì like rgb(255, 0, 0) (red).
h2 {
  color: red;                   /* color name */
}

.highlight {
  background-color: #ffeb3b;   /* hex color (a light yellow) */
}

button {
  background-color: rgb(0, 120, 215); /* rgb color (blue) */
  color: white;
}

6Ô∏è‚É£ The CSS box model: margin, border, padding

Every element on a web page is a ‚Äúbox‚Äù. The box has:

  • Content (text or image).
  • Padding (space around the content, inside the box).
  • Border (line around the box).
  • Margin (space outside the box, separating it from other boxes).
.card {
  background-color: #ffffff;         /* White background */
  padding: 20px;                     /* Space inside the box */
  border: 1px solid #dddddd;         /* Thin gray border */
  margin: 20px auto;                 /* Space outside, auto center */
  max-width: 600px;                  /* Maximum width of the card */
}
