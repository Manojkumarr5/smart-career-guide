In this stage you combine what you have learned so far (data types, loops, functions, files, error handling) to build two small but complete console projects: a Quiz App and a basic Calculator.

These projects help you think like a programmer: break a problem into steps, write clean functions, and interact with the user.


1Ô∏è‚É£ Project 1 ‚Äì Simple Quiz App (console)

This quiz runs in the terminal, asks multiple‚Äëchoice questions, checks answers, and shows your score at the end using a list of dictionaries for questions. 

# quiz_app.py

questions = [
    {
        "question": "1. Python is ______ language?",
        "options": ["a) compiled", "b) interpreted", "c) machine", "d) markup"],
        "answer": "b"
    },
    {
        "question": "2. Which data type is mutable?",
        "options": ["a) tuple", "b) string", "c) list", "d) int"],
        "answer": "c"
    },
    {
        "question": "3. Which keyword is used to define a function?",
        "options": ["a) fun", "b) def", "c) function", "d) lambda"],
        "answer": "b"
    },
    {
        "question": "4. Which module is commonly used for HTTP requests?",
        "options": ["a) math", "b) random", "c) requests", "d) os"],
        "answer": "c"
    },
]

def run_quiz(questions):
    print("=== Smart Career Guide ‚Äì Python Quiz ===")
    score = 0

    for q in questions:
        print()
        print(q["question"])
        for opt in q["options"]:
            print(opt)

        user_ans = input("Your answer (a/b/c/d): ").strip().lower()

        if user_ans == q["answer"]:
            print("‚úÖ Correct!")
            score += 1
        else:
            print("‚ùå Wrong. Correct answer was:", q["answer"])

    print("\nQuiz finished!")
    print("Your score:", score, "/", len(questions))

if __name__ == "__main__":
    run_quiz(questions)

Later you can move questions into a JSON file and load them dynamically, but for now keeping them in a list in the same file is simpler.

3Ô∏è‚É£ Project 2 ‚Äì Basic Calculator (console)

This calculator asks the user to choose an operation and enter two numbers, then prints the result using separate functions for each operation. 

# calculator.py

def add(a, b):
    return a + b

def sub(a, b):
    return a - b

def mul(a, b):
    return a * b

def div(a, b):
    if b == 0:
        return "Error: Cannot divide by zero"
    return a / b

def calculator():
    print("=== Simple Calculator ===")
    print("1) Add")
    print("2) Subtract")
    print("3) Multiply")
    print("4) Divide")

    choice = input("Choose operation (1/2/3/4): ").strip()

    if choice not in ["1", "2", "3", "4"]:
        print("Invalid choice.")
        return

    try:
        a = float(input("Enter first number: "))
        b = float(input("Enter second number: "))
    except ValueError:
        print("Please enter valid numbers.")
        return

    if choice == "1":
        result = add(a, b)
    elif choice == "2":
        result = sub(a, b)
    elif choice == "3":
        result = mul(a, b)
    else:
        result = div(a, b)

    print("Result:", result)

if __name__ == "__main__":
    calculator()

This is a console version; later you can build a GUI calculator using Tkinter or a web version using Flask or Django.

4Ô∏è‚É£ Combining both into a small menu app

You can put both mini projects into a single file with a simple menu so the user can choose ‚ÄúQuiz‚Äù or ‚ÄúCalculator‚Äù. 

def main_menu():
    while True:
        print("\n=== Mini Project Hub ===")
        print("1) Run Quiz")
        print("2) Open Calculator")
        print("3) Exit")

        choice = input("Choose an option (1/2/3): ").strip()

        if choice == "1":
            run_quiz(questions)
        elif choice == "2":
            calculator()
        elif choice == "3":
            print("Goodbye!")
            break
        else:
            print("Invalid choice, please try again.")

if __name__ == "__main__":
    main_menu()

This starts to feel like a tiny command‚Äëline app, and you can keep adding new tools such as a notes app, log viewer, or marks calculator.
