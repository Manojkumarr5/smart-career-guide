Step 3: Create Responsive Layouts (Flexbox & Grid)

In Stage 2, you learned how to style elements with CSS. In Stage 3, you will learn how to arrange those elements on the page so they look good on mobile, tablet, and desktop screens.

You will use Flexbox and CSS Grid to create flexible layouts, and a small media query to make them responsive.


1Ô∏è‚É£ What is a layout?

A layout is how elements are arranged on the page: side by side, in rows, in columns, or in a grid. In old websites, people used tables for layout, but now modern CSS uses Flexbox and Grid for this.

Responsive layout means the same page automatically adjusts to different screen sizes, without the user needing to zoom or scroll sideways.

You turn a container into a flex container using display: flex;, then control how the items are spaced using properties like justify-content and align-items.

3Ô∏è‚É£ CSS Grid basics (two-dimensional layout)

CSS Grid is used when you want to control both rows and columns at the same time. It is perfect for dashboards, galleries, or pages with multiple sections.

You turn a container into a grid using display: grid;, then define columns and rows using grid-template-columns and grid-template-rows.

4Ô∏è‚É£ Making layouts responsive with Flexbox

With Flexbox, you can allow items to wrap onto the next line using flex-wrap: wrap;. This means when the screen is narrow, some items can move to the next row automatically.

In a card layout, this helps cards stack nicely on smaller screens without extra work.

5Ô∏è‚É£ Media queries: change layout on small screens

Media queries let you apply different CSS rules when the screen is smaller than a certain width. This is how you adapt layouts for phones, tablets, and desktops.

In the example below, cards are 3 per row on big screens, 2 per row on tablets, and 1 per row on mobile.

/* Base: 3 cards per row (handled by flex: 1 1 calc(33.333% - 16px)) */


@media (max-width: 768px) {
  .responsive-card {
    flex: 1 1 calc(50% - 16px);   /* 2 cards per row on tablets */
  }
}


@media (max-width: 480px) {
  .responsive-card {
    flex: 1 1 100%;               /* 1 card per row on small phones */
  }

  .grid-two-column {
    grid-template-columns: 1fr;   /* Change grid to single column */
  }

  .flex-row {
    flex-direction: column;       /* Stack boxes vertically */
  }
}

6Ô∏è‚É£ When to use Flexbox vs Grid?

Use Flexbox when you mainly care about arranging items in a single row or column, such as navigation menus, button groups, or card rows.

Use Grid when you want a full layout with rows and columns, like dashboards, main content + sidebar + footer, or image galleries.
