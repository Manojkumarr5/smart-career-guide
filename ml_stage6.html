Step 6: Classification Algorithms

Decision Trees, Random Forest, and SVM are popular algorithms for classification tasks.


ðŸ“Œ Decision Tree

from sklearn.datasets import load_iris
from sklearn.tree import DecisionTreeClassifier

iris = load_iris()
X, y = iris.data, iris.target

clf = DecisionTreeClassifier()
clf.fit(X, y)

print("Prediction for first sample:", clf.predict([X[0]]))

ðŸ“Œ Random Forest

from sklearn.ensemble import RandomForestClassifier

rf = RandomForestClassifier(n_estimators=100)
rf.fit(X, y)

print("Prediction for first sample:", rf.predict([X[0]]))

ðŸ“Œ Support Vector Machine (SVM)

from sklearn.svm import SVC

svm_model = SVC()
svm_model.fit(X, y)

print("Prediction for first sample:", svm_model.predict([X[0]]))
