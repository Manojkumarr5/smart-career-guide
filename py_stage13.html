Step 13: Work with Flask or Django Basics

In this stage you meet Python web frameworks: Flask (lightweight) and Django (full‚Äëstack). You will build a tiny Flask app with routes and templates, and briefly see how Django structures a project.

This step connects your Python skills to real web pages that open in a browser, which is the base for APIs, dashboards, and full web apps.


1Ô∏è‚É£ Flask vs Django ‚Äì quick mental picture

Flask is a micro‚Äëframework that is small, flexible, and great for learning and small apps, while Django is a larger ‚Äúbatteries‚Äëincluded‚Äù framework with more built‚Äëin features.

  • Start with Flask if you want simple apps and to understand web basics step by step.
  • Move to Django when you want a more structured, larger application with built‚Äëin admin, auth, and more.

2Ô∏è‚É£ Install Flask and create first app

First, install Flask with pip.

# install once
pip install flask

Create a new file app.py and add:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello from Smart Career Guide!"

if __name__ == "__main__":
    app.run(debug=True)

@app.route("/") maps the URL / to the home function, and app.run(debug=True) starts the development server.

3Ô∏è‚É£ Run the Flask app

In a terminal, go to the folder where app.py is saved and run:

python app.py

Open the URL shown in the terminal (usually http://127.0.0.1:5000) in a browser to see the ‚ÄúHello from Smart Career Guide!‚Äù message.

4Ô∏è‚É£ Flask with templates (HTML files)

Instead of returning plain text, you normally return HTML templates from a templates folder.

Project structure:

your_folder/
    app.py
    templates/
        home.html

home.html (inside templates):




    
    Smart Career Guide ‚Äì Home


    
Welcome to Smart Career Guide

    
This page is rendered using a Flask template.

Update app.py to use render_template:

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template("home.html")

render_template searches the templates folder for the given file name and returns it as the HTTP response.

5Ô∏è‚É£ Simple multi‚Äëpage Flask app

You can add more routes and templates to create multiple pages such as home, about, and contact. 

@app.route("/about")
def about():
    return "
About
This is a small Flask demo app.
"

@app.route("/hello/<name>")

    return f"
Hello, {name}!
"

Dynamic routes like /hello/<name> let you capture parts of the URL as parameters.

6Ô∏è‚É£ Very quick look at Django structure (optional)

Django has a slightly heavier setup but is powerful for larger projects and follows a project/app structure.

# install once

Basic commands to start (just for awareness):

cd mysite
python manage.py runserver

Django organizes code into projects and apps and follows the MVT (Model‚ÄëView‚ÄëTemplate) pattern; at this stage, knowing the names and structure is enough.
