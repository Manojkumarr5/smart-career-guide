Step 3: Understand Statistics and Probability

Statistics and probability help you understand data distributions, variation, and uncertainty in ML.


ðŸ“Œ Descriptive Statistics

Mean, median, mode, variance, and standard deviation summarize data.

import numpy as np

data = np.array([10, 12, 15, 18, 20])

print("Mean:", data.mean())
print("Median:", np.median(data))
print("Std Dev:", data.std())

ðŸ“Œ Probability Basics

Probability measures how likely an event is to happen.

import random

trials = 1000
count_heads = 0

for _ in range(trials):
    flip = random.choice(["H", "T"])
    if flip == "H":
        count_heads += 1

print("Estimated P(Head):", count_heads / trials)

ðŸ“Œ Normal Distribution

Many real-world features (like height, marks) are approximately normally distributed.

import numpy as np

samples = np.random.normal(loc=50, scale=10, size=1000)
print("Sample mean:", samples.mean())
print("Sample std:", samples.std())
