Step 5: DOM Manipulation and Events

In Stage 4, you learned about JavaScript variables, loops, and functions. Now you will learn how to use JavaScript to actually change the page that the user sees: text, colors, elements, etc.

This is done through the DOM (Document Object Model) and events like clicks, input typing, and more.


1Ô∏è‚É£ What is the DOM?

DOM stands for Document Object Model. When the browser loads your HTML, it builds a tree-like structure in memory. Each HTML element becomes a JavaScript object that you can access and change.

You can select elements (like <p>, <div>, <button>) and change their text, HTML, styles, or even remove or add new elements.

2Ô∏è‚É£ Selecting elements: getElementById and querySelector

To change something, first you need to select it. Two very common methods are:

  • document.getElementById("someId") ‚Äì selects one element by its id.
  • document.querySelector("css-selector") ‚Äì selects the first element that matches a CSS selector.
// HTML

Welcome text goes here.



// JavaScript
const p = document.getElementById("welcomeText");
p.textContent = "Hello from JavaScript!";

3Ô∏è‚É£ Changing text and HTML content

You can change the inside of an element in two main ways:

  • textContent ‚Äì sets plain text (good when you just want text).
  • innerHTML ‚Äì sets HTML (text + tags).
// Using textContent
const msg = document.getElementById("msgText");
msg.textContent = "This is plain text.";


// Using innerHTML
const msg2 = document.getElementById("msgHtml");
msg2.innerHTML = "This has bold text using <strong>.";

4Ô∏è‚É£ Handling events with addEventListener

An event is something that happens in the browser: click, key press, mouse move, input change, etc. You can ‚Äúlisten‚Äù for an event and then run a function when it happens.

The addEventListener method connects an element to an event and a handler function.

// HTML





// JavaScript
const colorBtn = document.getElementById("colorBtn");
const colorBox = document.getElementById("colorBox");


colorBtn.addEventListener("click", () => {
  colorBox.style.backgroundColor = "tomato";
});

5Ô∏è‚É£ Reading user input and updating the page

You can get the value from an input field using the value property, and then show it somewhere else on the page.

This is how search bars, forms, and dynamic text work.

// HTML






// JavaScript
const nameInput = document.getElementById("nameInput");
const greetBtn = document.getElementById("greetBtn");
const greetOutput = document.getElementById("greetOutput");


greetBtn.addEventListener("click", () => {
  const name = nameInput.value;
  greetOutput.textContent = "Hello, " + name + "!";
});

6Ô∏è‚É£ Creating a simple todo list (add <li> items)

You can also create new elements with JavaScript and add them to the page. This is useful for lists, dynamic content, search results, etc.

Below is a simple todo list where each click on ‚ÄúAdd Task‚Äù adds a new list item.

// HTML



    


// JavaScript
const taskInput = document.getElementById("taskInput");
const addTaskBtn = document.getElementById("addTaskBtn");
const taskList = document.getElementById("taskList");


addTaskBtn.addEventListener("click", () => {
  const text = taskInput.value.trim();
  if (text === "") return;


  const li = document.createElement("li");
  li.textContent = text;
  taskList.appendChild(li);


  taskInput.value = "";
});

      7Ô∏è‚É£ Changing CSS styles with JavaScript

      You can change the style of an element directly from JavaScript using the style property. You write CSS property names in camelCase (for example, backgroundColor instead of background-color).

      Below is a button that toggles between light and dark mode for a small box.

      // HTML

      Light mode
      



// JavaScript
const modeBox = document.getElementById("modeBox");
const toggleBtn = document.getElementById("toggleModeBtn");
let isDark = false;


toggleBtn.addEventListener("click", () => {
  if (!isDark) {
    modeBox.style.backgroundColor = "#333";
    modeBox.style.color = "#fff";
    modeBox.textContent = "Dark mode";
    isDark = true;
  } else {
    modeBox.style.backgroundColor = "#f2f6ff";
    modeBox.style.color = "#000";
    modeBox.textContent = "Light mode";
    isDark = false;
  }
});
