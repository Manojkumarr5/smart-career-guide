Step 1: Install Python and IDE (VS Code, PyCharm)

This stage starts from zero. You will install Python on your system, verify that it works, and set up an editor (VS Code or PyCharm) so you can comfortably write and run Python programs.

Once this is done, you will be ready for all future Python stages (syntax, data types, projects, automation, web apps, and more).

Python is a high-level, general-purpose programming language used in web development, data science, automation, AI, and many other areas.

Python code is usually saved in files ending with .py and then executed by the Python interpreter installed on your machine.

2Ô∏è‚É£ Install Python (Windows)

Follow these basic steps on Windows.

  • Go to https://www.python.org/downloads/ and download the latest stable Python 3 installer for Windows.
  • Run the .exe installer.
  • On the first screen, tick ‚ÄúAdd Python to PATH‚Äù. This is important so you can run Python from any folder.
  • Click ‚ÄúInstall Now‚Äù and wait until the installation finishes.

Verify the installation using Command Prompt or PowerShell.

# Check Python version
python --version
or
py --version

You should see something like Python 3.x.x. If not, restart the terminal or check that Python was added to PATH.

3Ô∏è‚É£ Install Python (macOS)

On macOS, the recommended ways are the official installer or tools like Homebrew.

  • Option 1 ‚Äì Official installer: Go to https://www.python.org/downloads/macos/, download the latest macOS installer, run the .pkg file, and follow the wizard.
  • Option 2 ‚Äì Homebrew (for advanced users): install Homebrew, then run brew install python in the terminal.

Verify in Terminal:

# Check Python 3 version
python3 --version

On macOS, Python 3 is usually run with python3 instead of python.

5Ô∏è‚É£ Install VS Code for Python

VS Code (Visual Studio Code) is a lightweight, powerful editor that works well for Python.

  • Go to https://code.visualstudio.com/ and download the installer for your OS.
  • Install VS Code using the default options.
  • Open VS Code and install the ‚ÄúPython‚Äù extension by Microsoft from the Extensions panel.

Create and run a simple Python file in VS Code:

print("Hello, Python from VS Code!")

Then in VS Code:

  • Open the folder where hello.py is saved.[web:27]
  • Open hello.py and click the ‚ÄúRun‚Äù button, or open the built‚Äëin terminal and run:
python hello.py # Windows
python3 hello.py # macOS/Linux

You should see Hello, Python from VS Code! printed in the terminal.

6Ô∏è‚É£ Install PyCharm (optional but recommended)

PyCharm is a dedicated Python IDE with many helpful features for larger projects.

  • Go to https://www.jetbrains.com/pycharm/download/ and download the Community Edition (free).
  • Run the installer and follow the default steps for your OS.[web:28]
  • Open PyCharm and choose ‚ÄúNew Project‚Äù. Select a location and make sure the correct Python interpreter is chosen.

Create and run a simple script:

Run the file using the green ‚ÄúRun‚Äù button. Output should appear in the run console inside PyCharm.
