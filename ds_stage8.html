Step 8: Understand ML Models (Regression, Classification)

In this stage you meet the two main supervised learning families: regression (predict numbers) and classification (predict categories like pass/fail or yes/no).

You will see simple examples of linear regression and logistic regression using scikit‚Äëlearn so students can connect math and code to real predictions.


1Ô∏è‚É£ Regression vs classification ‚Äì simple idea

Regression problems predict a continuous number, such as marks, salary, or house price.

  • Example: ‚ÄúGiven study hours, predict the student‚Äôs marks out of 100.‚Äù
  • Example: ‚ÄúGiven house size and location, predict price in rupees.‚Äù

Classification problems predict a category/label, such as pass vs fail, spam vs not spam, or disease vs no disease.

  • Example: ‚ÄúGiven marks, decide pass/fail.‚Äù
  • Example: ‚ÄúGiven some features, decide whether a loan is approved or rejected.‚Äù

A common explanation is: regression ‚Üí continuous output; classification ‚Üí discrete class output.

2Ô∏è‚É£ Simple linear regression in code

Linear regression learns a straight‚Äëline relationship between input features and a numeric target such as scores. 

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

# Example: study_hours -> marks
data = {
    "study_hours": [1, 2, 3, 4, 5, 6],
    "marks":       [35, 40, 50, 55, 65, 70]
}
df = pd.DataFrame(data)

X = df[["study_hours"]]   # features (2D)
y = df["marks"]           # target (1D)

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

model = LinearRegression()
model.fit(X_train, y_train)

print("Slope (coefficient):", model.coef_)
print("Intercept:", model.intercept_)

predictions = model.predict(X_test)
print("Predicted marks:", predictions)

The model learns a line with a slope and intercept that minimizes the squared error between actual and predicted marks.

3Ô∏è‚É£ Simple logistic regression for binary classification

Logistic regression is a basic algorithm for binary classification when the target is a yes/no or 0/1 outcome. 

import numpy as np
from sklearn.linear_model import LogisticRegression

# Example: marks -> pass (1) / fail (0)
data = {
    "marks": [30, 35, 40, 50, 60, 70, 80],
    "pass":  [0, 0, 0, 1, 1, 1, 1]
}
df = pd.DataFrame(data)

X = df[["marks"]]
y = df["pass"]

log_reg = LogisticRegression()
log_reg.fit(X, y)

# Predict class (0 or 1)
print(log_reg.predict([[45], [65]]))

# Predict probability of passing
print(log_reg.predict_proba([[45], [65]]))

Logistic regression outputs probabilities between 0 and 1 and then assigns a class label using a cutoff such as 0.5.

4Ô∏è‚É£ How to choose: regression or classification?

A simple rule is to look at your target column and choose the type of model based on what you want to predict.

  • If the target is a number (marks, price, temperature) ‚Üí use regression.
  • If the target is a label (pass/fail, yes/no, A/B/C) ‚Üí use classification.

Some problems can be framed both ways (predicting a probability vs predicting a class), but this simple rule is enough at beginner level.
